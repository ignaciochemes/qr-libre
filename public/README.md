# Archivos Estáticos para SEO

Este directorio contiene los archivos estáticos necesarios para el SEO óptimo de QR Libre.

## Archivos Requeridos

### 1. og-image.png
- **Dimensiones:** 1200x630 píxeles
- **Formato:** PNG
- **Uso:** Imagen Open Graph para compartir en redes sociales
- **Contenido sugerido:** Logo de QR Libre + texto "Generar QR Online Gratis"

### 2. favicon.ico
- **Dimensiones:** 16x16, 32x32, 48x48 píxeles (múltiples tamaños en un archivo)
- **Formato:** ICO
- **Uso:** Favicon del sitio (aparece en la pestaña del navegador)

### 3. icon-192.png
- **Dimensiones:** 192x192 píxeles
- **Formato:** PNG
- **Uso:** Icono PWA para dispositivos Android

### 4. icon-512.png
- **Dimensiones:** 512x512 píxeles
- **Formato:** PNG
- **Uso:** Icono PWA para dispositivos Android (alta resolución)

### 5. apple-touch-icon.png
- **Dimensiones:** 180x180 píxeles
- **Formato:** PNG
- **Uso:** Icono para dispositivos iOS cuando se agrega a la pantalla de inicio

## Herramientas Recomendadas

Para crear estos archivos puedes usar:
- [Favicon Generator](https://realfavicongenerator.net/)
- [Canva](https://www.canva.com/) para crear og-image.png
- [Figma](https://www.figma.com/) para diseño de iconos

## Nota

Estos archivos son necesarios para un SEO completo. Mientras no existan, algunas funcionalidades de SEO no funcionarán al 100%, pero la aplicación seguirá funcionando correctamente.
